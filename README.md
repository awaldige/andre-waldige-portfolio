Bem-vindo ao meu laboratório de projetos! Este repositório contém o código-fonte do meu portfólio profissional, desenvolvido para apresentar minhas habilidades em desenvolvimento Full Stack, automação de processos e design de interfaces modernas.

🌐 **Veja o site no ar:** [https://andre-waldige.vercel.app](https://andre-waldige.vercel.app)

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído com foco em performance, SEO e experiência do usuário (UX), utilizando:

- **Frontend:** HTML5 Semântico, CSS3 Moderno (Flexbox/Grid), JavaScript Vanilla.
- **Backend & Database:** Node.js, PHP, MySQL, MongoDB.
- **Deploy & Analytics:** Vercel & Vercel Insights.
- **Design:** Inter Fonts, FontAwesome, Design Responsivo (Mobile First).

---

## 📂 Estrutura do Projeto

- `index.html`: Página principal com resumo profissional, serviços e stack técnica.
- `projetos.html`: Galeria completa e detalhada com todos os sistemas desenvolvidos.
- `css/style.css`: Estilização centralizada, animações e variáveis de design.
- `img/`: Assets visuais, ícones, favicons e o `banner-share.jpg` para redes sociais.
- `manifest.json`: Configurações de PWA (Progressive Web App).

---

## 🌟 Projetos em Destaque

Alguns dos sistemas que você encontrará detalhados no portfólio:

1. **Gerador de Contratos:** Automação de documentos jurídicos com preenchimento dinâmico.
2. **AutoManagerPro:** Sistema de gestão robusto integrado com MySQL para controle de frotas/veículos.
3. **TaskPad Kanban:** Gerenciador de tarefas interativo com persistência de dados.
4. **Streaming de Desenhos:** Aplicação moderna construída com Node.js e Express.
5. **Calculadora Fiscal:** Ferramenta complexa para cálculo de tributos (INSS/IRRF) atualizados.
6. **Sistema BiblioTech:** Painel administrativo para controle de acervos bibliográficos.
